# Dukaan - Premium E-commerce Website

Dukaan is a modern and fully responsive e-commerce website built using HTML, Tailwind CSS, and JavaScript.

This project is designed with a premium shopping UI and includes dynamic content rendering using JSON files. Products, categories, hero banners, testimonials, footer data, and newsletter content are all managed through JSON, making the project more scalable and easier to maintain.

The website includes features like product filtering, wishlist functionality, cart badges, auto-sliding hero banners, custom cursor effects, smooth animations, testimonial auto-scroll, newsletter section, and a modern footer.

This project is perfect for beginners who want to learn how to build a real-world frontend project using HTML, Tailwind CSS, JavaScript, and JSON data.

You can also use this project as a portfolio project, client project reference, or learning resource for frontend development.

## How To Run The Project

1. Clone the repository

```bash
git clone https://github.com/withaarzoo/Dukaan.git
```

2. Open the project folder

```bash
cd Dukaan
```

3. Install dependencies

```bash
npm install
```

4. Run Tailwind CSS build command

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./assets/output.css --watch
```

5. Open the project using Live Server

You can use the Live Server extension in VS Code to run the project locally.
